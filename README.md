# Dissertation Portfolio Website

Professional one-page portfolio website for **Nelis Munene**, focused on dissertation and thesis development support.

## Overview
This project presents academic consulting services with a clean, modern, and responsive layout. The site positions services ethically as refinement and research support (not ghostwriting).

## Tech Stack
- HTML5
- CSS3 (Grid/Flexbox)
- Vanilla JavaScript

## Key Sections
- Hero with profile image and focused CTAs
- Credibility stats strip
- Why Work With Me
- How I Support Your Dissertation
- Works Samples (with expandable abstracts + paper links)
- Client Testimonials
- Tools & Standards
- Contact section with email + WhatsApp options

## Contact Form
The contact form is wired to FormSubmit:
- Endpoint: `https://formsubmit.co/ajax/nellmunene44@gmail.com`
- Includes anti-spam honeypot and CAPTCHA
- Inline submission status feedback is handled in `script.js`

## Files
- `index.html` - page structure and content
- `styles.css` - visual design and responsive styles
- `script.js` - year auto-update + contact form submission/status handling

## Local Preview
Open `index.html` in a browser.

## Deployment
You can deploy this as a static site via:
- GitHub Pages
- Netlify
- Vercel

If using GitHub Pages, set the repository branch to `main` and root directory.
